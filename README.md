# 0. The Odin Project - Exercícios

Este repositório contém as soluções dos exercícios menores e desafios de lógica propostos pelo [The Odin Project](www.theodinproject.com). 

> **Nota:** Os projetos principais (como Landing Pages, Calculadoras, etc.) possuem repositórios próprios no meu perfil para melhor organização e deploy.

## 1. Tecnologias Praticadas
- HTML5 / CSS3
- JavaScript (Lógica e DOM)
- Git e GitHub

## 2. Organização
Os exercícios estão divididos por módulos conforme o currículo:
- `/html-css`: Práticas de estruturação, seletores e flexbox.
- `/javascript`: Fundamentos, funções, arrays e manipulação do DOM.
